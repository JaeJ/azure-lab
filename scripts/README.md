# Scripts

This folder contains PowerShell scripts used throughout the Azure Lab project.

## Included Scripts

### create-resource-group.ps1

Creates an Azure Resource Group.

---

### list-azure-resources.ps1

Lists Azure resources.

---

### cleanup-resource-group.ps1

Removes Azure lab resources.

---

## Requirements

- Azure Subscription
- Azure PowerShell Module
- PowerShell 7

Authenticate first:

```powershell
Connect-AzAccount
```
